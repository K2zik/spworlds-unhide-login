# SP Worlds — показать кнопки Discord/Telegram

Показывает скрытые по умолчанию кнопки входа через **Discord** и **Telegram** в окне авторизации на сайте [spworlds.ru](https://spworlds.ru).

## Описание

На сайте sp worlds в окне входа изначально видна только кнопка входа через Госуслуги, а кнопки Discord и Telegram скрыты через CSS-класс `hidden`. Этот скрипт убирает данный класс, делая обе кнопки видимыми и кликабельными.

Скрипт отслеживает изменения DOM (сайт построен на SvelteKit и рендерит контент динамически), поэтому кнопки остаются видимыми даже после переходов внутри сайта без полной перезагрузки страницы.

## Что делает скрипт

- Ищет кнопки с текстом `Discord` и `Telegram` в окне входа
- Убирает у них класс `hidden`
- Следит за динамическими изменениями страницы через `MutationObserver`, чтобы кнопки не скрывались повторно

## ⚠️ Важно

Скрипт влияет **только на отображение** кнопок и не добавляет никакой новой функциональности. Если вход через Discord или Telegram не реализован на стороне сайта (не активирован на бэкенде), нажатие на кнопку может не сработать или вернуть ошибку. Скрипт не взаимодействует с серверами Discord/Telegram и не собирает никаких данных.

## Совместимость

Работает в браузерах с поддержкой Tampermonkey / Violentmonkey: Chrome, Firefox, Edge, Opera и др.
